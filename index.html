Common Questions

How long does setup take?
Most firms are processing documents within 48 hours. Self-serve signup takes 5 minutes. First automation typically runs on day one.
What documents can it process?
Invoices, contracts, delivery notes, bills of lading, purchase orders, permits, quotes, and receipts. Email classification works with any business communication.
Do we need technical staff to run this?
No. The system runs in Docker containers with a simple web interface. If you can use email, you can use this.
What happens if the AI gets something wrong?
Every output includes a confidence score. Low-confidence results are flagged for human review. You set the threshold—we recommend 0.7 for most use cases.
Can we try it before committing?
Yes. Demo mode lets you test with your real documents. No credit card required. No time-limited trial pressure.
